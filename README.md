# Khalil Nooh Site

Astro content site for Khalil Nooh's professional profile, portfolio, public posting timeline, blog, and mirrored media library.

## Commands

```bash
npm install
npm run dev
npm run build
npm run crawl:dry
```

## Content

- `src/content/posts` stores timeline and blog entries.
- `src/content/projects` stores portfolio cards.
- `src/content/media` stores mirrored public media metadata.
- `src/content/sources` stores public source configuration.

The crawler creates reviewable drafts or source stubs. It does not use authenticated scraping.
